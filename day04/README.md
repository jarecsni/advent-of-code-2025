# Day 4: Printing Department

## Problem

The printing department has rolls of paper (@) arranged on a grid. Forklifts can only access a roll if there are **fewer than 4** rolls in the 8 adjacent positions (Moore neighborhood).

## Example

```
..@@.
@@@@.
 @@@.@.
@.@@@
 @@@@@.@.
@@
 @.@@@@..@.
@@.@@@@.@@
.@@@@@@@.@
.@.@.@.@@@
@.@@@.@@@@
.@@@@@@@@.
@.@.@@@.@.
```

In this example, 13 rolls are accessible (marked with x in the solution).

## Task

Count how many rolls of paper can be accessed by a forklift.
